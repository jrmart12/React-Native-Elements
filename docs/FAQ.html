When buying React Native Elements, how will I be updated? And how can I report issues?

Once you acquire React Native Elements please send us your Github username. We will grant you access to the repository where the starter kit is develop and you will get notified for each update. You will also be able to use the Github issues to ask questions and report potential issues.
